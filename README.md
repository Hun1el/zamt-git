## 📋 Описание

Репозиторий содержит материалы и задания по дисциплине **«Инструментальные средства разработки программного обеспечения**.

В рамках предмета рассматривались основные инструменты разработки и работы с проектами.

## 📚 Темы курса

* работа с системой контроля версий **Git**;
* управление репозиториями;
* работа с ветками и историей изменений;
* основы работы с **Docker**;
* контейнеризация приложений.
